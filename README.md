# nblog-app-webui
博客应用前台webui
基于vue-cli
克隆本项目到本地然后通过vscode打开，再执行以下命令
## 项目安装
```
npm install
```

### 编译热加载项目（用于测试开发）
```
npm run serve
```

### 编译构建压缩项目（用于生成环境）
```
npm run build
```

### Lints and fixes files 
```
npm run lint
```

### 自定义配置
See [Configuration Reference](https://cli.vuejs.org/config/).
